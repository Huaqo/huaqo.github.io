## Creating a Virtual Environment

```bash
# Create a virtual environment
python -m venv myenv
```

## Activating a Virtual Environment

### On Unix or macOS

```bash
# Activate the virtual environment
source myenv/bin/activate

# Deactivate the virtual environment
deactivate
```
